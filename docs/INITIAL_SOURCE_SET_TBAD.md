# Initial TBAD and Preemptive TEVAR Source Set

**Date registered:** 2026-07-15

## Purpose

This initial collection establishes a bibliographic foundation for questions concerning type B aortic dissection (TBAD) and preemptive thoracic endovascular aortic repair (TEVAR). It is intentionally limited to registered metadata and does not contain synthesized recommendations, numerical outcomes, or claim-level citations.

## Collection Composition

The set contains four guidelines to establish United States, Japanese, European, and multinational guidance coverage. Three randomized-trial publications represent the initial INSTEAD report, its long-term follow-up, and ADSORB. One prospective registry adds the VIRTUE multicenter cohort. Two observational or secondary analyses add natural-history and imaging-predictor perspectives without treating those analyses as guideline recommendations.

Bibliographic metadata has been registered, but source content has not yet been specialist validated. Recommendation wording and numerical outcomes must be checked against the original source before use in generated answers. No copyrighted PDFs were downloaded or committed.
