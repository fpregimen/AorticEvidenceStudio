# Private Local Review Copies

This directory is reserved for locally held source documents used during manual content review. Document files here are ignored by Git and must not be committed.

- No patient-identifiable data may be stored here.
- Copyright and redistribution rights must be respected.
- A filename should begin with the registered `source_id`.
- The source catalog must never expose this private local path in the browser.
- Do not place a PDF, DOC, DOCX, PPT, or PPTX here unless local possession and review are permitted.

This README is the only tracked file intended for this directory. Do not add actual source documents to the repository.
